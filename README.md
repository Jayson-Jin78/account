# 재정 관리 웹앱 사용 가이드

## 웹앱 실행 방법

### 방법 1: 로컬에서 실행
1. `webapp` 폴더를 찾습니다
2. `index.html` 파일을 더블클릭하여 브라우저에서 엽니다
3. 또는 브라우저에서 파일 → 파일 열기 → `index.html` 선택

### 방법 2: 안드로이드 태블릿에서 실행
1. `webapp` 폴더 전체를 구글 드라이브에 업로드
2. 태블릿에서 구글 드라이브 앱 열기
3. `index.html` 파일을 Chrome 브라우저로 열기

## 주요 기능

### 📊 재정 현황 탭
- **요약 정보**: 총 자산, 이번 달 수입/지출/순수익 표시
- **저축 자산 구성 차트**: 진아와 용근의 저축 상품을 누적 막대로 표시
- **월별 지출 차트**: 최근 6개월 카테고리별 지출을 누적 막대로 표시

### 💸 지출 관리 탭
- **월별 × 카테고리별 테이블**: 최근 6개월 지출 내역
- **직접 편집**: 셀을 클릭하여 금액 수정 가능
- **자동 합계**: 월별 총액과 카테고리별 총액 자동 계산

### 💰 저축 관리 탭
- **저축 상품 목록**: 모든 저축 상품 정보 표시
- **현재 평가액**: 원금 + 현재까지 발생한 이자
- **예상 이자**: 만기까지의 예상 이자
- **이자소득세**: 일반과세 상품의 세금 자동 계산
- **행 추가/삭제**: 새 저축 상품 추가 또는 기존 항목 삭제

## 현재 제한사항 (샘플 데이터 모드)

⚠️ 이 버전은 **프로토타입**으로 샘플 데이터를 사용합니다.

- 데이터는 브라우저를 새로고침하면 초기화됩니다
- 지출 관리 탭에서 셀 편집은 가능하지만 저장되지 않습니다
- 저축 관리 탭에서 행 추가/삭제는 임시로만 작동합니다

## 다음 단계: 구글 시트 연동

2단계에서 구글 시트와 연동하면:
- ✅ 데이터가 구글 시트에 자동 저장됩니다
- ✅ 여러 기기에서 동일한 데이터를 볼 수 있습니다
- ✅ 구글 시트에서도 직접 편집 가능합니다
- ✅ 자동 백업 및 버전 관리

## 모바일 사용 팁

- **가로 모드**: 테이블을 볼 때는 태블릿을 가로로 회전하세요
- **확대/축소**: 두 손가락으로 핀치하여 확대/축소 가능
- **스크롤**: 테이블은 좌우로 스크롤 가능합니다

## 문제 해결

### 차트가 표시되지 않는 경우
- 인터넷 연결을 확인하세요 (Chart.js 라이브러리 로드 필요)
- 브라우저를 새로고침하세요

### 한글이 깨지는 경우
- Chrome 브라우저를 사용하세요
- 파일 인코딩이 UTF-8인지 확인하세요
